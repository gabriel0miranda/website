Transparência Web

Acho que quando falamos de transparência alguns detalhes técnicos ficam de fora, detalhes esses que são indispensáveis ao passo que o mundo se digitaliza cada vez mais. A transparência governamental brasileira, por exemplo, está distante de ter mecanismos efetivos para realmente mostrar o que acontece por dentro dos palácios e das câmaras, mesmo sendo completamente possível implementar tecnologias que facilitam a visualização pública, via internet, de dados tangentes à sociedade.

Neste texto pretendo abordar a transparência de hospedagem de serviços, ferramentas e páginas na internet.

Basicamente o caso é o seguinte: eu, empresa/organização/governo, tenho um serviço/ferramenta/página na web e gostaria de oferecer aos usuários informações sobre determinados elementos de interesse, como de onde estão sendo feitos os acessos ao site em cada momento, qual software está sendo utilizado para fazer o site funcionar, como está a saúde de uma carteira de doações que mantem o host do site pago, etc.
